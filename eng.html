Mermaid 순서도

sequenceDiagram
    participant User
    participant MCP_Server
    participant Resolver
    participant LLM
    participant Crawler
    participant KG
    participant RAG

    User->>MCP_Server: Send question + digital link
    MCP_Server->>Resolver: Request list of link types from digital link
    Resolver-->>MCP_Server: Return list of link types

    MCP_Server->>LLM: Send question + list of link types
    LLM-->>MCP_Server: Return Top-N relevant link types

    loop For each selected link type
        MCP_Server->>Crawler: Crawl data from link
        Crawler-->>MCP_Server: Return cleaned data
    end

    MCP_Server->>KG: Query knowledge graph with question
    KG-->>MCP_Server: Return related triples or info

    MCP_Server->>RAG: Send question + crawled data + KG data
    RAG-->>MCP_Server: Generate response

    MCP_Server-->>User: Return final response
