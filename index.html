Mermaid 순서도



```mermaid
sequenceDiagram
    participant User
    participant MCP_Server
    participant Resolver
    participant LLM
    participant Crawler
    participant KG
    participant RAG

    User->>MCP_Server: 질문 + 디지털 링크 전달
    MCP_Server->>Resolver: 디지털 링크로 링크타입 목록 요청
    Resolver-->>MCP_Server: 링크타입 목록 반환

    MCP_Server->>LLM: 질문 + 링크타입 목록 전달
    LLM-->>MCP_Server: 관련 링크타입 Top-N 반환

    loop 선택된 링크타입마다
        MCP_Server->>Crawler: 해당 링크로 접속하여 크롤링
        Crawler-->>MCP_Server: 정제된 데이터 반환
    end

    MCP_Server->>KG: 질문과 관련된 지식요청
    KG-->>MCP_Server: 관련 트리플 or 정보 반환

    MCP_Server->>RAG: 질문 + 크롤링 데이터 + KG 데이터 전달
    RAG-->>MCP_Server: 응답 생성

    MCP_Server-->>User: 최종 응답 반환
```
